# my-first-site

金沢市公式ウェブサイトをイメージした静的なランディングページです。近代的なデザインと伝統的な金沢らしさを調和させたヒーローセクション、暮らし・手続き、最新のお知らせ、観光・文化体験などのコンテンツを紹介しています。

## 特長

- **伝統とモダンの融合**: 金沢らしい配色とタイポグラフィを採用し、上質さと洗練さを両立させたデザイン。
- **レスポンシブレイアウト**: CSS Grid と Flexbox を活用し、PC・タブレット・スマートフォンで快適に閲覧できるレイアウトを実現。
- **豊富なセクション構成**: 暮らしや行政手続きの案内、最新ニュース、文化体験の紹介など、市民・観光客双方に役立つ情報を想定して配置。
- **アクセシビリティへの配慮**: 対比の高いカラーリングやセマンティックな HTML を用いて、読みやすさと操作性を高めています。

## ファイル構成

```
my-first-site/
├── index.html   # メインの静的ページ
└── README.md    # 本ドキュメント
```

## セットアップと閲覧方法

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/<your-account>/my-first-site.git
   cd my-first-site
   ```
2. ブラウザで `index.html` を開きます。
   ```bash
   open index.html        # macOS
   xdg-open index.html    # Linux
   start index.html       # Windows (PowerShell)
   ```

## カスタマイズのヒント

- **配色やフォントの調整**: `<style>` ブロックの `:root` で CSS カスタムプロパティを定義しているので、ブランドカラーの変更が容易です。
- **セクションの追加**: `<main>` 内の `<section>` をコピーして必要なコンテンツを追加できます。必要に応じて `.grid` や `.experience` といったユーティリティクラスを再利用してください。
- **コンポーネントの再利用**: `.card` や `.news-item` などのスタイルは他のページでも流用しやすいよう汎用的に設計されています。
